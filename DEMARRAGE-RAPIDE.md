# ⚡ Démarrage rapide du serveur PWA iPad

## 🚀 Méthodes de lancement (par ordre de simplicité)

### 1. ✅ **Double-clic sur "PWA iPad Server.command"**
- **Nouveau fichier créé spécialement pour vous !**
- Double-cliquez et ça fonctionne immédiatement
- S'ouvre automatiquement dans Terminal

### 2. ✅ **Application macOS : "PWA iPad Server.app"**
- Double-cliquez sur l'application
- Interface native macOS

### 3. ✅ **Via Terminal (méthode sûre)**
```bash
# Ouvrir Terminal et taper :
cd ~/Documents/GitHub/NBLI
./launch-server.sh
```

## 🛠 Problème résolu : fichiers .sh qui s'ouvrent dans Xcode

**Le problème :** macOS associe parfois les fichiers `.sh` à Xcode au lieu du Terminal.

**Solutions :**
- ✅ **Utilisez "PWA iPad Server.command"** (recommandé)
- ✅ **Utilisez "PWA iPad Server.app"**
- 🔧 **Changez l'association :** Clic droit → Obtenir des infos → Ouvrir avec Terminal

## 📱 Une fois le serveur démarré

L'adresse IP sera affichée dans Terminal. Exemple :
```
📱 Application accessible sur iPad à l'adresse:
   http://10.0.0.105:8000
```

Copiez cette adresse dans Safari sur votre iPad !

---

**🎯 Conseil :** Utilisez **"PWA iPad Server.command"** - c'est la solution la plus simple et la plus fiable !
